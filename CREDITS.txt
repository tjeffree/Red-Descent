RED DESCENT — Asset Credits & Licenses
=======================================

This file tracks the provenance and licensing of all third-party assets.
CC0 assets require no attribution, but are listed here for documentation and
to protect against future license disputes. See
docs/Red_Descent_Asset_Sourcing_Guide.md for selection rationale.

DERIVED — Martian terrain block tiles
  Files  : assets/generated/mars_dirt.png, mars_rock.png, mars_basalt.png,
           mars_permafrost.png, mars_ore.png  (build script:
           _tools/make_mars_tiles.py)
  Source : procedurally baked from hand-tuned Martian palettes (rusty
           regolith -> volcanic basalt -> dusty subsurface ice). The ore tile
           composites the CC0 Kenney cyan gem (tile 67, Pixel Platformer) onto
           the Mars host rock for high-contrast vein readability.
  License: CC0 1.0 (original generated art; gem overlay is a CC0 derivative)
  Used   : superseded by the sliced block-tile set below (kept for reference;
           mars_dirt.png remains the surface ground-plane fallback)

DERIVED — Block tiles (7 types x 9 variants)
  Files  : assets/generated/tiles/<type>_<0..8>.png  for type in
           dirt, rock, basalt, permafrost, ore, bulkhead, vault
           (build script: _tools/slice_tiles.py)
  Source : sliced from the author-supplied sheet assets/many tiles.png — a
           9x7 grid of framed blocks, one type per row, 9 variants each. The
           slicer crops each tile to its solid square, inpaints the rounded
           black corners from neighbouring colour, and downscales to 32x32.
  License: Project author's own asset (all rights reserved to the author).
  Used   : all terrain block types, with per-cell variant variety on the 3D
           dig cubes (Dirt/Rock/Basalt/Permafrost/Ore + Ruins Bulkhead/Vault)

DERIVED — Surface ship (4 repair stages)
  Files  : assets/generated/wreckage-0.png .. wreckage-3.png
           (build script: _tools/make_wreckage.py)
  Source : background-masked + commonly-cropped from the project author's own
           illustrations assets/wreckage-sprite.jpg (stage 0 = crashed wreck)
           and assets/wreckage-sprite-repair-1..3.jpg (stage 3 = fully
           repaired); grey field flood-filled to alpha, hull-aligned crop.
           Originals generated by the author with Google Gemini.
  License: Project author's own asset (all rights reserved to the author).
  Used   : the mother ship on the surface, swapped by repair progress
           (Phase 6 ship-repair track)
